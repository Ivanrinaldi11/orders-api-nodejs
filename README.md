## 🚀 API de Pedidos

🔗 Repositório: https://github.com/Ivanrinaldi11/orders-api-nodejs

⚠️ Este projeto é uma API backend e não possui interface visual.

### ▶️ Como testar

Utilize ferramentas como Postman ou Insomnia para testar os endpoints:

- GET /orders  
- POST /orders  
- PUT /orders/:id  
- DELETE /orders/:id
